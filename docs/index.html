How to use the framework?

require the following to load the framework: 

(require '[vvvvalvalval.supdate.api :as supd :refer-macros [supdate]])
Let's define the rules of the game: 

(def data
  {:a 1
   :b [{:c 42}
       {:c 43}
       {:c-is-not "here"}]})

(supd/supdate
  data
  {:a inc
   :b [{:c dec}]})
